#### Rappels de 1ere NSI
| Commande | Exemple d'utilisation | Explication |
    | :--- | :--- | :--- |
    | **`cat`** | `cat file.txt` | Affiche tout le contenu d'un fichier. |
    | **`cd`** | `cd /home` | **C**hange **D**irectory : navigue vers le dossier indiqué. |
    | **`chmod`** | `chmod +x script` | **Ch**ange **Mod**e : modifie les permissions (ex: rendre exécutable). |
    | **`clear`** | `clear` | Efface l'écran du terminal. |
    | **`cp`** | `cp a.txt b.txt` | **C**o**p**y : copie un fichier (`-r` pour un dossier). |
    | **`echo`** | `echo \"Salut"\` | Affiche du texte (souvent utilisé pour écrire dans un fichier avec `>`). |
    | **`grep`** | `grep \"mot"\ f.txt` | Recherche un texte spécifique à l'intérieur d'un fichier. |
    | **`history`**| `history` | Affiche la liste des dernières commandes tapées. |
    | **`id`** | `id` | Affiche les identifiants (UID, GID) et les groupes de l'utilisateur actuel. |
    | **`ifconfig`**| `ifconfig` | Affiche la configuration réseau (ancien outil, déprécié mais courant). |
    | **`ip`** | `ip addr` | Affiche/configure les interfaces réseau (remplace *ifconfig*). |
    | **`less`** | `less gros.log` | Affiche le contenu page par page (`q` pour quitter). |
    | **`ls`** | `ls -la` | **L**i**s**te les fichiers/dossiers (`-la` pour les détails et fichiers cachés). |
    | **`mv`** | `mv a.txt b.txt` | **M**o**v**e : déplace ou renomme un fichier/dossier. |
    | **`mkdir`** | `mkdir photos` | **M**a**k**e **Dir**ectory : crée un nouveau dossier. |
    | **`more`** | `more fichier.txt` | Affiche le contenu page par page (ancêtre de `less`, plus limité). |
    | **`nano`** | `nano file.txt` | Ouvre un éditeur de texte simple directement dans le terminal. |
    | **`ping`** | `ping google.fr` | Vérifie la connectivité vers une adresse IP ou un site web. |
    | **`pwd`** | `pwd` | **P**rint **W**orking **D**irectory : affiche le chemin du dossier actuel. |
    | **`rmdir`** | `rmdir vide` | **R**e**m**ove **Dir**ectory : supprime un dossier **uniquement s'il est vide**. |
    | **`rm`** | `rm fichier.txt` | **R**e**m**ove : supprime un fichier (`-r` pour un dossier et son contenu). |
    | **`route`** | `route -n` | Affiche la table de routage (ancien outil). |
    | **`touch`** | `touch notes.txt` | Crée un fichier vide ou met à jour sa date. |
    | **`tree`** | `tree` | Affiche l'arborescence des fichiers et dossiers sous forme d'arbre visuel. |
    | **`sudo`** | `sudo apt update` | Exécute la commande en tant qu'administrateur (SuperUser). |
    | **`traceroute`**| `traceroute 8.8.8.8` | Affiche le chemin (les routeurs) emprunté par les données. |
    | **`wc`** | `wc -l file.txt` | **W**ord **C**ount : compte les lignes (`-l`), mots ou caractères. |
    | **`which`** | `which python` | Localise et affiche le chemin complet de l'exécutable d'une commande. |
    | **`whoami`** | `whoami` | Affiche le nom de l'utilisateur actuellement connecté. |\n
